使用说明：
1. 本目录下包括CMakeLists.txt, shell脚本run.sh， 命令行版本的可执行文件：task_master, GUI版本的可执行文件：日程管理大师.exe, 命令行版本的源代码（包含在finalProject文件夹），以及GUI版本的源代码（包含在）
2. 本项目命令行版本的可执行文件为task_master，运行方式为：linux终端在该文件所在目录下输入./task_master run运行。
3. GUI版本的可执行文件为：日程管理大师.exe，运行方式为，windows下双击打开。
4. linux下的脚本测试文件为run.sh，自动对添加、删除、查询任务的功能进行测试。可以修改其中每一行参数的前两个为任意一个已注册账号的ID、密码，从而对相应账号进行任务操作的测试。运行方式为：linux终端在该文件所在目录下输入./run.sh运行。
